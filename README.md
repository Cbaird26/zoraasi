# Zora Discovery

Zora Discovery is a visual form explorer organized around the Zora Pattern.

It focuses on:

- a layered toroidal pattern
- a wide conceptual scale keyboard from `10^-122` to `10^122`
- a calm two-stroke pulse with long release
- direct user control over geometry, drift, and bloom

This repo is intentionally visual-first. It does not include the old run-corpus analyzer or native mobile/desktop layers.
